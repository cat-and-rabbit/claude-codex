Claude Code は、Anthropic が提供する AI コーディングアシスタントである。

## チャットとの違い

Claude.ai のようなチャットサービスとは、できることが根本的に異なる。

チャットでは、テキストをやり取りするだけである。コードの断片を貼り付けて相談することはできるが、Claude がファイルを読みにいくことも、変更を加えることもできない。

Claude Code は違う。ファイルを直接読み、書き、コマンドを実行する。自分のコードベースに入り込んで作業する点がチャットとの本質的な違いである。

## 何ができるか

- ファイルを読んで内容を把握する
- ファイルを新規作成・編集する
- ターミナルでコマンドを実行する
- エラーが出たら原因を調べて修正する
- 複数のファイルにまたがる変更を行う

これらをひとつの会話の中で連続して行う。「このバグを直してほしい」と頼めば、コードを読み、修正し、テストを走らせ、結果を報告するまでを自律的に進める。

## エージェントとして動くということ

Claude Code は指示に受動的に答えるのではなく、目標を達成するために自分で考えて動く。こうした AI を「エージェント」と呼ぶ。

ただし、何でも任せてよいというわけではない。Claude Code は間違えることがある。変更の内容を確認し、意図と違う動きをしていればすぐに止める習慣を持つことが重要である。エージェントを使いこなすとは、信頼しつつも目を離さないということでもある。
